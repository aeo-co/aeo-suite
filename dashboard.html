How to get a clean PDF:
1. Click Generate PDF — your browser's print dialog opens.
2. Set Destination to "Save as PDF".
3. Open More settings and uncheck "Headers and footers".
4. Set Background graphics to on so the dark theme renders.
5. Set Margins to "Default" or "Minimum".
6. Click Save.
