How to get a clean PDF:
1. Click Generate PDF — your browser's print dialog opens.
2. Set Destination to "Save as PDF".
3. Open More settings and uncheck "Headers and footers" — this removes the URL, date, and page-number lines.
4. Set Background graphics to on so the dark theme renders correctly.
5. Set Margins to "Default" or "Minimum" for best fit.
6. Click Save.
